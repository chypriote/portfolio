Verbex

Verbex is a web app, made to make simple regular expressions a bit easier to construct, by using word-blocks, and by highlighting matches on predefined text. It's built on top of JSVerbalExpressions, using jQuery. Verbex can be found at madskonradsen.dk/verbex
